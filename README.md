### About

- This repository includes the sampled data used for the experiments in the paper "Does Using Counterfactual Help LLMs Explain Textual Importance in Classification?".
- There are 3 sampled datasets:
  - Amazon: `amazon_random300.csv`
  - SST2: `SST2_random300.csv`
  - IMDB: `IMDB_random300.csv`
